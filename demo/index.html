Features

  • Easy to customize

    Text works like any regular Zdog shape, you can control the color, size, alignment, etc and update them at any time

  • Use any font

    Zfont is built on top of Typr.js, which provides solid support for a wide range of .ttf and .otf fonts

  • Helpful extras

    A small collection of extra utilities help with measuring text, waiting for fonts to load, and more!
